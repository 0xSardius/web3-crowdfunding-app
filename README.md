# Web3 Crowdfunding App

## Description

This is a web3 crowdfunding app built with Next.js, Tailwind CSS, and Thirdweb. It allows users to create and manage campaigns, and to contribute to campaigns.

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
